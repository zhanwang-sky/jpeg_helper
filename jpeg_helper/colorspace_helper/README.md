##### RGB => YCbCr

```
Y = a * R + b * G + c * B
Cb = (B - Y) / d
Cr = (R - Y) / e
```

##### YCbCr => RGB

```
R = Y + e * Cr
G = Y - (a * e * Cr) / b - (c * d * Cb) / b
B = Y + d * Cb
```
